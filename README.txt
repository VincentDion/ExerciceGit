Le premier commit de ce projet porte sur la création de ce fichier README.

Le second commit porte sur le contenu de ce fichier.

Le troisième commit portera sur la création et le contenu du fichier intitulé helloWorld.py.

Le quatrième commit portera sur la création et un premier jet de contenu pour le fichier intitulé aboutMe.txt

Le cinquième commit portera sur une modification du fichier intitulé aboutMe.txt
